# Spaceable

## Installation

In your Gemfile:

    gem 'spaceable', :git => 'git://github.com/cicatriz/spaceable.git'

In the User model:

    acts_as_learner

In the model of the thing you want to space:

    acts_as_component

Install and migrate:

    rails g spaceable:install
    rake db:migrate
